Windows 11 安装 Miniconda 完全指南

结合官方指南与社区经验优化的详细安装流程

Miniconda 是什么？

Miniconda 是 Anaconda 的轻量级版本，包含 Conda 包管理器和 Python，可高效管理 Python 虚拟环境及科学计算包。本指南将带您完成从下载到优化配置的全过程。

1

下载 Miniconda 安装包

选择版本

  • 访问 Miniconda 官网 或 清华镜像源
  • 选择 Windows 64位版本 (如 Miniconda3-latest-Windows-x86_64.exe)

下载注意事项

避免使用浏览器内置下载工具，推荐使用迅雷或IDM加速下载

2

安装 Miniconda

启动安装程序

双击下载的 .exe 文件，以管理员身份运行安装程序

关键安装选项

安装用户选择

  • 勾选 "Just Me" (仅当前用户)
  • 若误选"All Users"，后续需手动修改配置文件

安装路径设置

默认路径为 C:\Users\<用户名>\Miniconda3

建议更改为非系统盘 (如 D:\Miniconda3)

环境变量配置

  • 勾选 "Add Miniconda to PATH" (全局可用)
  • 或选择 "Register Anaconda as default Python"
  • 推荐前者，便于多Python版本共存

完成安装

等待安装进度条完成，最后取消勾选 "Launch Miniconda" (避免自动进入base环境)

3

配置环境变量 (若未自动添加)

手动添加路径

  1. 1 按 Win + S 搜索"环境变量"
  2. 2 打开"系统属性 → 高级 → 环境变量"
  3. 3 在 "系统变量" 中找到 Path，点击编辑 → 新建
  4. 4 添加以下路径 (根据实际安装路径调整)：
D:\Miniconda3
D:\Miniconda3\Scripts
D:\Miniconda3\Library\bin

验证安装

打开命令提示符 (CMD) 或 PowerShell，输入：

conda --version
python --version

显示版本号即表示安装成功

4

优化配置

更换国内镜像源 (加速包下载)

配置清华源 (推荐)：

conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/free/
conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/main/
conda config --set show_channel_urls yes

验证配置：

conda config --show channels

取消自动激活 base 环境

执行以下命令避免启动终端时自动进入 base 环境：

conda config --set auto_activate_base false

需要时可手动通过 conda activate 激活环境

解决 PowerShell 权限问题 (可选)

若使用 PowerShell 报错"无法加载 profile.ps1"，以管理员身份运行：

Set-ExecutionPolicy RemoteSigned

输入 Y 确认策略更改

5

常见问题处理

虚拟环境路径异常

若误选"All Users"，需手动创建 .condarc 文件：

在 C:\Users\<用户名> 下新建 .condarc 文件，写入：

envs_dirs:
  - D:\Miniconda3\envs # 自定义环境存储路径

修改安装目录权限：右键 Miniconda 文件夹 → 属性 → 安全 → 编辑，赋予当前用户"完全控制"权限

conda 命令无效

检查环境变量路径是否正确添加，或重新初始化：

conda init cmd.exe
6

验证安装成果

创建测试虚拟环境：

conda create -n test_env python=3.9
conda activate test_env
pip install numpy
python -c "import numpy; print(numpy.__version__)"

成功输出版本号即表示配置完整
